# The Practice of Paying Attention

**One-line definition.**

## Summary
Attention is the foundational practice behind art and writing; it can be trained like a muscle.

## Key points
- Attention is a pre-thought state of noticing the world directly. ([[2026-06-29--the-practice-of-paying-attention]] )
- Exercises like blind contour drawing and walking without purpose strengthen attention. ([[2026-06-29--the-practice-of-paying-attention]] )
- Attention is an ethical gesture and a resistance to distraction. ([[2026-06-29--the-practice-of-paying-attention]] )
