# The Sketchbook Habit

**One-line definition.**

## Summary
A daily sketchbook practice frees the writer from self-editing and deepens visual attention.

## Key points
- Sketching gives permission to be bad and reduces the pressure on first drafts. ([[2026-06-29--the-sketchbook-habit]] )
- Drawing trains more granular seeing that transfers to writing. ([[2026-06-29--the-sketchbook-habit]] )
- A simple daily routine makes the habit sustainable. ([[2026-06-29--the-sketchbook-habit]] )
