# Revision as Sculpting

**One-line definition.**

## Summary
Revision is a subtractive process like sculpture: remove everything that is not the poem.

## Key points
- The first draft is raw material containing the poem and its excess. ([[2026-06-29--on-revision-as-sculpting]] )
- Good revision removes whole sections to reveal the poem's true shape. ([[2026-06-29--on-revision-as-sculpting]] )
- Know when to stop before over-polishing the work away. ([[2026-06-29--on-revision-as-sculpting]] )
